# Filth-4
A open source compression tool that is easy and reliable.
# How to use?
## Installing
### 1 - First, clone the project on git. Go to Filth 4's main directory and run:
```
chmod +x filth4.sh
```
### And after that, run the filth4.sh file. With that, you can now use Filth 4 globally on your enviroment.

## Commands
### Compress: use filth4 -c desired_output.f4 desired_file.jpg desired_file.img and etc to compress your files.
### Extract: use filth4 -x gaven_output.f4 to extract your filth 4 file.
### Display: use filth4 -d gaven_output.f4 to display the files contained in the .f4 file.
